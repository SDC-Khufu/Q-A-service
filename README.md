# QnA-service

## Description

In order to optimize the backend for shopping site: Hyper Fashion. The backend got rebuilt with three services: products, QnA, Review. This QnA service contains server routes to interact with client and database for over 1 million products. It is built with Node.js, Express, Node-Postgres, PostgreSQL, and containerized with docker. 

## Getting Started

### Prerequisites

You will need to install the following:

```
npm
node.js
```

### Installing

```
git clone repo
npm install
npm start
open localhost:3555
```

## Running the tests

```
npm test
```

## Deployment

This can be deployed using AWS.
